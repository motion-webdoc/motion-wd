1.HTML негиздер

Документтин негизги элементи болуп саналат. Бардык башка элементтер < html >...< /html > ичинде камтылган. Элементтин сыртындагы эч нерсе браузер тарабынан HTML коду катары кабыл алынбайт жана эч кандай жол менен иштетилбейт..

Бөлүм < head >...< /head > барак жөнүндө техникалык маалыматты камтыйт: аталышы, сүрөттөмөсү, издөө системалары үчүн ачкыч сөздөр, коддоо ж. Ага киргизилген маалымат браузердин терезесинде көрсөтүлбөйт, бирок браузерге баракты кантип иштетүү керектигин айткан маалыматтарды камтыйт.

< head > бөлүмдүн милдеттүү элементи болуп < title > саналат. Текст, < title > элементтин ичине жайгаштырылат, ал браузердин аталыш тилкесинде көрсөтүлөт.

Аталышка толук дал келүү үчүн аталыштын узундугу 60 белгиден ашпашы керек. Титулдук текст мүмкүн болушунча веб -баракчанын мазмунун толук сыпаттамасын камтышы керек..

Кошумча < head > бөлүм элементи < meta > болуп саналат. Анын жардамы менен сиз баракчанын мазмунун жана издөө системалары үчүн ачкыч сөздөрдү, HTML документинин авторун жана башка метадайындардын касиеттерин орното аласыз.

Элемент < head >бир нече < meta > элементтерди камтышы мүмкүн, анткени, колдонулган атрибуттарга жараша, алар ар кандай маалыматты алып жүрүшөт.

Бул элементтин ичинде страницада колдонулган стилдер коюлат. CSS, HTML документин стилдөө үчүн колдонулат. Баракта мындай бир нече элементтер болушу мүмкүн..

Сиз дагы башка ыкманы колдонуу менен документтин стилдерин орното аласыз - аларды .css кеңейтүүсү менен өзүнчө файлга жазыңыз, мисалы, style.css.

Файлды стили бар веб -баракка туташтыруунун эки жолу бар.:
@import url директива аркылуу
< link > элементин колдонуу. Бул ыкма артыкчылыктуу болуп саналат.

< link > элементи учурдагы барак менен башка документтердин ортосундагы байланышты аныктайт. Баракта мындай бир нече элементтер болушу мүмкүн.

< script > элементи документиңизге ар кандай скрипттерди тиркөөгө мүмкүндүк берет. Скрипт текстик сценарияны бул элементтин ичинде же тышкы файлдарда жайгашышы мүмкүн. Эгерде скрипт текстик сценариянын тышкы файлылында жайгашкан болсо, анда ал элементтин атрибуттарын колдонуу менен кошулат.

< body > бөлүмү документтин бүт мазмунун камтыйт.

2.HTML-тегдер

HTML тегдери HTML тилинин негизи болуп саналат. Тегтер белгилөөдө элементтердин башын жана аягын чектөө үчүн колдонулат.

Ар бир HTML документи HTML элементтеринин жана тексттин дарагынан турат. Ар бир HTML элементи баштоо (ачуу) жана аяктоо (жабуу) теги менен аныкталат. Баштоо жана аяктоо тегдери тегдин атын камтыйт.

Бардык HTML элементтери беш түргө бөлүнөт:

  • бош элементтер — < area >, < base >, < br >, < col >, < embed >, < hr >, < img >, < input >, < link >, < menuitem >, < meta >, < param >, < source >, < track > , < wbr >;
  • форматталбаган тексти бар элементтер — < script >, < style >;
  • форматталбаган текстти чыгаруучу элементтер — < textarea >, < title >;
  • башка аталыш мейкиндигиндеги элементтер — MathML жана SVG;
  • кадимки нерселер — башка бардык нерселер.

3.HTML - атрибуттар

HTML атрибуттары браузерге тигил же бул барактын элементин кантип көрсөтүү керектигин айтат. Атрибуттар бир эле тегдердин жардамы менен кошулган маалыматтын көрүнүшүн ар түрдүү кылууга мүмкүндүк берет.

Атрибут мааниси тырмакчага алынат " ". Атрибуттардын аталыштары жана баалуулуктары чоң -кичине сезгич эмес, бирок ошентсе да аларды кичине тамга менен терүү сунушталат.

Глобалдык атрибуттар
Төмөндөгү таблицада көрсөтүлгөн глобалдык атрибуттар каалаган HTML элементи үчүн колдонулушу мүмкүн, бирок алардын айрымдары элементтерге эч кандай таасирин тийгизбеши мүмкүн.

Учурдагы нерсеге жетүү үчүн баскычтардын жарлыктарын түзөт. Боштук менен бөлүнгөн белгилердин тизмесинен турат. Браузер алгач клавиатуранын жайгашуусунда бар баскычтарды тандайт. Төмөнкү элементтерге колдонулат: <a>, <area>, <button>, <input>, <label>, <legend>, <textarea>. Кабыл алынган баалуулуктар: ачкыч аттардын тизмеси.

Элементтин класс аталышын аныктайт (стилдер барагындагы классты аныктоо үчүн колдонулат). Кабыл алынган баалуулуктар: класстын аты.

Колдонуучу мазмунун (мазмунун) түзөтө аларын аныктайт. Сиз каалаган HTML талаасын түзөтүлүүчү элементке айландырууга мүмкүнчүлүк берет. Кабыл алынган баалуулуктар: true/false.

<bdo> жана <bdi> элементтериндеги мазмун текстинин багытын аныктайт. Кабыл алынган баалуулуктар: ltr / rtl / auto.

Колдонуучу элементти сүйрөй аларын аныктайт. Кабыл алынган баалуулуктар: true / false / auto.

Элементтин жашырылышы керек экенин көрсөтөт. Кабыл алынган баалуулуктар: hidden.

Бир нерсе үчүн уникалдуу идентификаторду аныктайт. Кабыл алынган баалуулуктар: id - элементтин идентификатору.

Элементтеги мазмундун (мазмундун) тилдик кодун аныктайт. Кабыл алынган баалуулуктар: тил коду.

Элементтин мазмуну орфографиялык жана грамматикалык текшерүү экендигин көрсөтөт. Кабыл алынган баалуулуктар: true/false.

Элементти стилдөө үчүн колдонулган CSSти көрсөтөт. Кабыл алынган баалуулуктар: CSS коду.

TAB баскычын колдонуу менен элементке өтүү тартибин аныктайт. Кабыл алынган баалуулуктар: сериялык номер.

Барак үчүн кеңештерди коюу менен элемент жөнүндө кошумча маалыматты аныктайт. Кабыл алынган баалуулуктар: текст.

Элемент ичиндеги текстти которууну иштетет же өчүрөт. Кабыл алынган баалуулуктар: yes/no.
